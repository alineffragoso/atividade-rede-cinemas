# atividade-rede-cinemas

Projeto desenvolvido para a disciplina de Engenharia de Software.

## Objetivo

Desenvolver um sistema simples para gerenciamento de rede de cinemas utilizando:

- UML
- MVC
- Service
- Repository
- SQLite

## Funcionalidades

- Cadastro de filmes
- Cadastro de sessões
- Registro de público
- Consulta de filmes
- Consulta de sessões
